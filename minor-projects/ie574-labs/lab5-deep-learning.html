What Did I Do?

In this capstone lab, I taught a robot to think like a human. Using artificial intelligence and neural networks, I created a system where the robot can look at an object, identify what it is (carton, glass, metal, or plastic), and automatically place it in the correct bin—all without being explicitly programmed for each type of object.

Goal: Train a deep learning model to classify 4 different material types with 97%+ accuracy, then use it to automatically sort 12 mixed objects.

The Challenge

Instead of writing rules like "if the object is shiny, it's metal," I let the robot learn patterns from examples. This is the power of artificial intelligence:

  • Train a neural network on hundreds of labeled images
  • The network learns what makes each material type unique
  • Use the trained model to classify new, unseen objects
  • Achieve high accuracy (97%+) with minimal manual programming

The AI Brain: Neural Network Architecture

I designed a deep neural network with multiple layers to learn increasingly complex patterns:

Network Layers

  • Convolution Layer 1: 16 filters to detect basic features (edges, textures)
  • Pooling Layer 1: Reduces size while keeping important information
  • Convolution Layer 2: 32 filters to detect more complex patterns
  • Pooling Layer 2: Further simplification
  • Convolution Layer 3: 64 filters to detect high-level features
  • Pooling Layer 3: Final size reduction
  • Dense Layer: 128 neurons to make the final classification decision

Why this design? Each layer learns different levels of complexity. Early layers learn simple features (edges), middle layers learn combinations (shapes), and deep layers learn high-level patterns (material types).

Optimization & Tuning

To achieve 97%+ accuracy, I made several key improvements:

Architecture Improvements

  • Added extra convolution layers for deeper learning
  • Increased filters per layer (16 → 32 → 64) to extract richer features
  • Expanded dense layer from 64 to 128 units for better classification

Training Parameters

  • 100 training epochs to ensure convergence
  • Batch size of 32 for stable gradient updates
  • Adam optimizer with default learning rate (0.001)
  • Categorical cross-entropy loss for 4-class classification
