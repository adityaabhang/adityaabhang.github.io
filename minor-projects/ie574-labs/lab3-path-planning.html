Path Planning with Smooth Curves

📅 Spring 2026
🏫 Lab 3
🤖 Purdue University
Path Planning Robotics Motion Control Trajectory Design

Project Setup

For this lab, I worked with a UR3 robot in a simulated manufacturing environment. The goal was to plan smooth, collision-free paths for the robot to pick up objects and place them in different locations.

What Did I Do?

In this lab, I designed smooth paths for a robot to follow. Instead of moving the robot in straight lines (which can be jerky and inefficient), I created curved paths that allow smooth, natural movement.

Goal: Plan 4 different paths for the robot to complete 2 pick-and-place tasks while avoiding obstacles and other robots.

The Challenge

The robot needed to:

  • Pick up an object from one location
  • Navigate around obstacles and another robot safely
  • Place the object in a drop-off location
  • Do this twice with different paths

How Path Planning Works

Path planning involves breaking a complex movement into smaller, manageable steps:

1
Define Key Points
Mark the start, goal, and waypoints the robot must pass through
2
Create Smooth Curves
Connect points with smooth mathematical curves instead of straight lines
3
Check for Collisions
Verify the path doesn't hit any obstacles or other objects
4
Execute the Path
Send the smooth trajectory to the robot to follow

Smooth Curves in Path Planning

I used two mathematical methods to create smooth paths:

Ferguson Curves (for paths 1 & 2)

A type of curve that smoothly connects two points while maintaining direction control. These curves are like drawing a smooth arc between points instead of a straight line.

Bézier Curves (for paths 3 & 4)

Another smooth curve method that uses control points to shape the path. This gives more flexibility in how the robot moves through space.

Why use curves? Curves create smooth, natural movement. Robots moving along curves instead of jagged lines are faster, smoother, and less likely to damage sensitive equipment.

The Four Paths

Path 1: Approach to Pick Location

The robot moves from its starting position to the first object. The path carefully descends while avoiding obstacles.

Path 2: Move to First Drop Location

After picking up the object, the robot travels to the first drop-off point, moving around the KUKA robot in the workspace.

Path 3: Return to Pick Second Object

The robot moves back to the workspace to pick up a second object, taking a different route to avoid interference.

Path 4: Move to Second Drop Location

Finally, the robot delivers the second object to its destination, completing the task.

Results

✅ 4 Smooth Paths Planned

Successfully created 4 collision-free paths with smooth curves connecting 4 points each

🎯 No Collisions

All paths avoided obstacles and the other KUKA robot working in the shared space

⚡ 44 Total Waypoints

Generated 11 intermediate points per path (4 paths × 11 points = 44 waypoints)

📊 Two Methods Compared

Used Ferguson and Bézier curves to understand different path generation techniques

What I Learned

  • Motion Planning: How to design robot movements that are smooth, efficient, and collision-free
  • Workspace Analysis: Understanding how to work with multiple robots in shared environments
  • Curve Mathematics: How Ferguson and Bézier curves create smooth trajectories
  • Waypoint Design: The importance of intermediate points (waypoints) in path planning
  • Practical Robotics: That good path planning is crucial for safe, efficient automation

Why This Matters

Path planning is essential in real manufacturing environments:

  • Manufacturing Lines: Multiple robots work together safely by planning collision-free paths
  • Efficiency: Smooth curves reduce energy use and make robots work faster
  • Safety: Careful planning prevents robots from hitting each other or obstacles
  • Automation: Good path planning allows factories to run complex multi-robot operations 24/7
  • Precision: Smooth paths lead to more accurate object placement
