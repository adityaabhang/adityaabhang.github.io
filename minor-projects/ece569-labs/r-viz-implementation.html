Overview

This lab focused on 3D visualization of robotic systems using RViz. I loaded robot models from URDF descriptions, visualized the TF (Transform) tree showing coordinate frame relationships, and displayed simulated sensor data including joint states and markers. RViz is the primary tool for debugging and monitoring robot state in ROS-based systems.

Methodology

I worked with URDF robot descriptions to load and display robot models in RViz, configured the TF tree to visualize kinematic chains, added interactive markers for end-effector targets, and set up real-time joint state visualization. The lab demonstrated how to use RViz as a debugging and development tool for robotic applications.
