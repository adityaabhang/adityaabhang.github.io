Overview

This lab introduced the fundamentals of the Robot Operating System (ROS). I set up a complete ROS workspace, created custom nodes for publishing and subscribing to topics, implemented service-client communication, and built launch files to orchestrate multi-node robotic systems. The lab covered the core messaging infrastructure that underpins all modern robotic software stacks.

Methodology

Starting with workspace setup and catkin build system configuration, I progressed through creating publisher and subscriber nodes in Python, implementing custom message types, and building service servers. The lab culminated in a multi-node system that coordinated sensor data flow between simulated robot components.
