Overview

At Purdue's IDEAS Lab, I'm developing a fully autonomous drone system capable of navigating through dense forest canopies while tracking human subjects in real-time. The project addresses a critical gap in search-and-rescue and environmental monitoring: most existing drone systems fail in heavily occluded, GPS-denied environments where tree cover disrupts both navigation and target visibility.

The system combines a robust computer vision pipeline with LiDAR-based obstacle avoidance, multi-object tracking, and a multi-agent communication protocol that relays live target coordinates from a scouting drone to a parent UAS for continuous situational awareness.

Methodology

The pipeline is built around three core subsystems that work in concert:

1. Perception & Obstacle Avoidance: The drone fuses LiDAR point cloud data with RGB camera feeds to build a real-time 3D map of the environment. LiDAR handles structural obstacles (trees, branches, terrain changes) while the RGB stream provides texture-level detail for gap detection and path validation. The obstacle avoidance module processes this fused data to generate safe flight corridors through the canopy.

2. Human Tracking Pipeline: For target tracking, I evaluated multiple multi-object tracking (MOT) approaches including ByteTrack and BoT-SORT with camera motion compensation (CMC). The tracking system needs to handle frequent full occlusions as subjects move behind trees, variable lighting from canopy filtering, and appearance changes. ReID-capable tracking ensures re-identification after prolonged occlusion events, which is critical in forest scenarios where a subject can disappear behind cover for several seconds.

3. Multi-Agent Communication: The scouting drone transmits real-time target coordinates, confidence scores, and trajectory predictions to a parent UAS operating at a higher altitude. This layered approach ensures continuous coverage even when the scout loses line-of-sight, and allows the parent to coordinate broader area search patterns.

Challenges

Dense occlusion handling: Forest environments create persistent, unpredictable occlusions. A target can be fully visible one frame and completely hidden the next as they move behind a tree trunk. Standard trackers lose identity quickly in these conditions. We addressed this by integrating ReID features that maintain identity across multi-second occlusion gaps, and by using trajectory prediction to estimate target position during blind periods.

Variable lighting conditions: Canopy-filtered sunlight creates extreme dynamic range — bright patches of direct light next to deep shadow. This wreaks havoc on both detection confidence and appearance-based ReID. We implemented adaptive exposure management and lighting-invariant feature representations to maintain tracking consistency.

Real-time constraint on embedded hardware: The entire pipeline needs to run on drone-mounted compute (Jetson platform) with strict latency requirements. Teammate Ayush Kumar is running a parallel YOLO + BoT-SORT + Eigen Trajectories pipeline on a Jetson, and we're benchmarking throughput vs. accuracy tradeoffs to find the optimal operating point for real-time flight.
